
|      Notations      |              Key               | Symbol |        Value         |
| :-----------------: | :----------------------------: | :----: | :------------------: |
|      `<S-...>`      |           shift-key            |   ⇧    |     shift \`<S-`     |
|      `<C-...>`      |   control-key or command key   |   ^    |  control ctrl `<C-`  |
| `<M-...>`/`<A-...>` |      alt-key or meta-key       |   ⎇    | meta alt `<M-`/`<A-` |
|      `<D-...>`      |  command-key (Macintosh only)  |   ⌘    |        `<D-`         |
|      `<t_xx>`       | key with "xx" entry in termcap |        |                      |
